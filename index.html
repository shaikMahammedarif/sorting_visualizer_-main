Bubble Sort

Bubble Sort is a simple sorting algorithm that repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order.

Use Cases

  • Educational purposes
  • Small datasets
  • Nearly sorted arrays

Pros and Cons

  • Pro: Easy to understand and implement
  • Con: Inefficient for large datasets

Time Complexity

  • Best Case: O(n)
  • Average Case: O(n²)
  • Worst Case: O(n²)

Selection Sort

Selection Sort divides the input list into two parts: a sorted portion and an unsorted portion. It repeatedly selects the smallest element from the unsorted portion and adds it to the sorted portion.

Use Cases

  • Small datasets
  • When memory usage is a concern

Pros and Cons

  • Pro: Simple and performs well on small lists
  • Con: Inefficient for large datasets

Time Complexity

  • Best Case: O(n²)
  • Average Case: O(n²)
  • Worst Case: O(n²)

Quick Sort

Quick Sort is a highly efficient sorting algorithm that uses divide-and-conquer to sort elements. It is much more efficient on large lists compared to algorithms like insertion sort.

Use Cases

  • Large datasets
  • Unsorted arrays
  • External sorting (when the entire data set can't be loaded into memory)

Pros and Cons

  • Pro: Very efficient for large datasets
  • Con: Performance degrades for nearly sorted datasets or when choosing poor pivots

Time Complexity

  • Best Case: O(n log n)
  • Average Case: O(n log n)
  • Worst Case: O(n²)
